# Magic Item Tools
*Source: Dungeon Master's Guide (2024) p. 220* 

**Magic Item Tools**

| Item Category | Required Tool |
|---------------|---------------|
| Armor | [Leatherworker's Tools](compendium/5e/items/leatherworkers-tools.md), [Smith's Tools](compendium/5e/items/smiths-tools.md), or [Weaver's Tools](compendium/5e/items/weavers-tools.md) depending on the kind of armor as noted in the tools' descriptions |
| Potion | [Alchemist's Supplies](compendium/5e/items/alchemists-supplies.md) or [Herbalism Kit](compendium/5e/items/herbalism-kit.md) |
| Ring | [Jeweler's Tools](compendium/5e/items/jewelers-tools.md) |
| Rod | [Woodcarver's Tools](compendium/5e/items/woodcarvers-tools.md) |
| Scroll | [Calligrapher's Supplies](compendium/5e/items/calligraphers-supplies.md) |
| Staff | [Woodcarver's Tools](compendium/5e/items/woodcarvers-tools.md) |
| Wand | [Woodcarver's Tools](compendium/5e/items/woodcarvers-tools.md) |
| Weapon | [Leatherworker's Tools](compendium/5e/items/leatherworkers-tools.md), [Smith's Tools](compendium/5e/items/smiths-tools.md), or [Woodcarver's Tools](compendium/5e/items/woodcarvers-tools.md) depending on the kind of weapon as noted in the tools' descriptions |
| Wondrous Item | [Tinker's Tools](compendium/5e/items/tinkers-tools.md) or the tool required to make the nonmagical item on which the magic item is based |
^magic-item-tools
