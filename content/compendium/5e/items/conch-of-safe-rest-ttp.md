# Conch of Safe Rest


- **Weight**: 2.0 lbs.

You can use an action to blow this pearlescent conch like a [horn](compendium/5e/items/horn-phb.md), creating an effect identical with that of a [Leomund's tiny hut](compendium/5e/spells/leomunds-tiny-hut.md) spell. The conch disappears once the spell takes effect. The effect ends if you leave the spell's area.

*Source: The Tortle Package p. 17*
