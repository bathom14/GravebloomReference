# Bewitching Whispers
*Source: Player's Handbook p. 110. Available in the <span title='Systems Reference Document (5.1)'>SRD</span>*  

**Prerequisite**: Level 7

You can cast [compulsion](compendium/5e/spells/compulsion.md) once using a warlock spell slot. You can't do so again until you finish a long rest.
